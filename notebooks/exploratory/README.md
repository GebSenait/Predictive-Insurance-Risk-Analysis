# Exploratory Analysis Notebooks

This directory contains Jupyter notebooks for exploratory data analysis.

## Usage

1. Start Jupyter Lab:
   ```bash
   jupyter lab
   ```

2. Open notebooks from this directory

3. Ensure dependencies are installed:
   ```bash
   pip install -r requirements/dev.txt
   ```

## Notebook Naming Convention

- `01_data_loading.ipynb` - Initial data loading and exploration
- `02_data_cleaning.ipynb` - Data cleaning and preprocessing
- `03_eda.ipynb` - Exploratory data analysis
- `04_feature_engineering.ipynb` - Feature engineering
- `05_modeling.ipynb` - Model development

## Best Practices

- Clear markdown sections explaining analysis
- Comments in code cells
- Save outputs appropriately
- Document findings and insights
